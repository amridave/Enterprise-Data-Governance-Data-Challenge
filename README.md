# Enterprise-Data-Governance-Data-Challenge
My submission for the January 2026 FP20 Data challenge on Enterprise Data Governance.
The challenge was to step into the role of a Data Governance Analyst or Anlytics leader whose mandate is to identify governance gaps and design a practical, business-ready framework.
The goal is to improve data trust, enable safe and responsible data usage, strengthen data culture and support scalable confident decision making across the organization.
